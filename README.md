# BwAI-GDG-Career-Agent-Workshop-Demo

To run this on your system, do the following:
1. Create a venv using "python -m venv venv"
2. Activate it using "source ./venv/bin/activate" (or equivalent command for Windows)
3. Install requirements.txt using "pip install -r requirements.txt"
4. Put in your API key in the API_KEY cell of the career_agent_demo_v2.ipynb file.
5. Test out the Career Agent using the sample data provided!
